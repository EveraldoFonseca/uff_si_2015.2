Número: 03
Enunciado: Um supermercado deseja conceder um desconto para as compras com valor maior do que certo limite. Esse limite corresponde ao valor para o qual a quantidade de compras com valor acima dele é menor do que 20% da quantidade total de compras. Escreva um programa em Java para ler arquivos de dados de compras, imprimir uma tabela de quantidade percentual de compras em dez faixas e o valor do limite de desconto. A primeira faixa corresponde à quantidade de compras com valores maiores do que o menor valor de compra. A segunda faixa corresponde à quantidade de compras com valores maiores do que menor valor + (maior valor – menor valor)/10. A terceira faixa corresponde à quantidade de compras com valores maiores do que menor valor + 2*(maior valor – menor valor)/10 e assim por diante. Os dados encontram-se armazenados em um arquivo com três colunas código da compra (inteiro positivo), quantidade do item (inteiro positivo) e valor unitário do item (real positivo). O arquivo está ordenado pelo código da compra. Utilize as classes Compra e Item. A primeira deve possuir como atributos o código da compra e uma coleção de itens. A segunda deve possuir a quantidade e o valor unitário. A classe Compra deverá implementar método para calcular o total da compra e o item método para calcular o valor do item. Ambas as classes deverão possuir construtores para inicialização dos seus atributos. A classe compra deverá inicializar somente o seu atributo código e a classe Item deverá inicializar os seus dois atributos. A classe compra deverá possuir método para adicionar um novo item à sua coleção de itens.Considere que cada compra possui no máximo dez itens e que no arquivo de dados essa quantidade não é excedida.Os dados deverão ser lidos do arquivo e armazenados em memória para só então o programa calcular os valores pedidos. Deverá ser utilizada a seguinte função auxiliar. public static float[] histograma(Compra[] compras, float menorValor, float maiorValor)
